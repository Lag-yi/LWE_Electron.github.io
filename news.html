记录「神途」模组从零到一的每一步

归档

  • 2026年
2026.04.04
🎉 圣化盐与水晶瓶 · 生日特别更新

添加了「圣化盐」和「水晶瓶」及其合成配方。这天是 Starry_Electron 的生日，祝生日快乐！🎂

圣化盐可通过下界石英 + 荧石粉合成，用于后续进阶合成。水晶瓶用于存储圣水（计划中）。

2026.03.08
⚔️ 盔甲系统入门 · 海魂盔甲纹理绘制中

学会了自定义盔甲材料（ArmorMaterial），正在为海魂盔甲绘制纹理。下一步实现全套盔甲的特殊效果（例如水下呼吸+速度）。

2026.03.01
🛡️ 战神徽章完成 · 死亡触发13种增益

战神徽章效果：手持时死亡会阻止死亡，恢复1生命值，清除负面效果，并给予13种正面状态（速度、力量、抗性、隐身等），持续40秒。消耗一枚徽章。

代码已整合进主类，通过事件监听实现。

2026.02.25
🏗️ 模组基础框架搭建

创建 Fabric 1.21.1 项目，注册了第一个自定义物品「海魂锭」，编写了有序合成配方。

搭建了 GitHub 仓库和 GitHub Pages 官网，实现深浅主题切换和手机响应式菜单。
